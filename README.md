## Comandos básicos

#### Primeiro Passo:

 - Confirgurar .env

#### Segundo Passo:

- Instalar as dependências

* Ambiente de Desenvolvimento:

```
  npm i 
```

#### Terceiro Passo:

- Criar base de dados Postgres: src/infra/databases/scripts

#### Quarto Passo:

- Rodar a aplicação

```
  npm run dev 
```

#### Quinto Passo:

- Importar a collection "collection_fc_backend.json" no Insomnia.
